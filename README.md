\# Blood Bank Management System



This project is a Database Management System (DBMS) based Blood Bank Management System.



\## Technologies Used

\- MySQL

\- AngularJS

\- Node.js

\- Express.js

\- HTML

\- CSS



\## Features

\- Donor Registration

\- Blood Inventory Management

\- Blood Availability Search

\- Recipient Blood Requests

\- Issue Blood to Recipients



\## Project Structure



backend

Contains Node.js server code



frontend

Contains website pages (HTML, CSS, AngularJS)



SQL

Contains database creation scripts and queries



Report

Contains the DBMS project report



\## How to Run



1\. Start MySQL

2\. Import SQL files

3\. Run backend server



node server.js



4\. Open index.html using Live Server
